# `gui-src/`

TypeScript sources for the Receipt Board GUI. Bundled with `esbuild` into
`src/receipt_board/gui/static/` (the assets loaded in the `pywebview` window).

The build step and commands are added in issue
[#8](https://github.com/automatix/receipt-board/issues/8). See `docs/TECH_SPEC.md` §7.
